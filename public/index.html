What is Towers of Orul ?

This project began 3 years ago when the studio was created but it did not get the success we expected. We decided to launch the project again with a whole new team.

Orul is an RPG in a world of fantasy. You will first get to personnalize your character and explore this whole new world on your own. Unique quests, vast exploration, NPC interactions and a whole lot of features awaits you in this FPS oriented game. If you prefer multiplayer games then you can bring someone to play with you thanks to our multiplayer features !

Considered solutions :

  • Disable all VirtualBox network interfaces to prevent them from interfering with the game
  • Godot's Navigation Agent 3D for the pathfinding
  • Setting a maximum value for the vertical axis for the camera
